## DeepSeek4Mac
This application is created using Electron JavaScript Framework.
### App Features:
- Use <b>CONTROL+SPACE</b> to display app window.</br>
- Change models (DeepSeek / OpenAI)</br>
- Stay in tray, undisturbing workspace.
